¿Se pueden ver las citas pasadas o futuras?

No, no se guarda ningun registro dentro de este sitio web que pueda verificar su citas pasadas o futuras

¿Comó puedo agendar una cita?

Una vez dentro de nuestro sitio web dirijase a la sesión agendar citas, rellenando los campos correspondientes a la información personal, Luego en la sesión de agendar cita colocar rigurosamente el servicio, el doctor, la fecha y hora en la que puede estar disponible una vez realizado esto se debe enviar la solicitud.

¿Se puede agendar citas para un familiar o amigo?

Si, se puede agendar una cita para un familiar o amigo, siempre y cuando la información suministrada en el sitio web sea precisa

¿Como realizó la cancelación del servicio?

Para cancelar un servicio, debes dirigirte personalmente a las instalaciones de la fundación, específicamente a caja, para realizar el trámite correspondiente.
